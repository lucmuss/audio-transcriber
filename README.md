# audio-transcriber
Audio-transcriber ist ein schlankes, aber leistungsstarkes Kommandozeilentool, das die Spracherkennung (Speech-to-Text) vereinfacht. Es ermöglicht die nahtlose Transkription von Audiodateien direkt von der Kommandozeile aus, indem es sowohl die OpenAI Whisper API als auch lokale Ollama-Modelle unterstützt.
